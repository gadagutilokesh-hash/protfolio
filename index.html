My Projects

LearnLoop

An innovative learning platform designed to provide personalized education experiences. Built with Streamlit, the platform offers a comprehensive suite of features including curated course recommendations, interactive learning environments, skill testing, code compilation with AI assistance, PDF document interaction, personalized roadmaps, and certificate generation. LearnLoop leverages modern technologies such as Google's Generative AI, FAISS vector storage, and Cassandra database to create a seamless, user-friendly learning ecosystem that adapts to individual learning styles and goals.

CostLens

A comprehensive cost-of-living comparison platform that helps users make informed decisions about relocations and lifestyle changes. The application allows users to compare living expenses across different cities, including housing, transportation, food, and utilities. Built with modern web technologies, CostLens provides real-time data analysis and interactive visualizations to help users understand if a job offer, city move, or lifestyle change actually makes financial sense for them.

Certificate - Case of Living Situation
